# App 美化方案

## 设计主题

以 DeepSeek 品牌意象为基础，将「鲸鱼」图标进行娘化（Q 版拟人化），打造统一、可爱、清新的蓝色系视觉风格。

## 1. App 图标

- **背景**：纯白底，可带极浅蓝色投影或纯白无投影。
- **主体**：一只可爱的鲸鱼娘图案，色调以 DeepSeek 蓝（#0D7377 附近）为主。
- **形象要求**：
  - Q 版拟人化鲸鱼女孩，表情可爱、友好。
  - 保留鲸鱼尾巴、喷水孔等鲸鱼特征。
  - 可加入蝴蝶结、小帽子、眼睛高光等萌化元素。
  - 图案居中，占比 60%–70%，留有安全边距。
- **输出格式**：矢量 SVG 或高清 PNG；Android 项目最终放入 `res/mipmap-xxxhdpi/` 与 `res/drawable/`。

## 2. App 页面

- **主色调**：以图标同款蓝色为品牌色，整体蓝色风格。
- **页面背景**：浅蓝白渐变或纯白，避免高饱和荧光蓝。
- **余额卡片**：可使用蓝色渐变或蓝色容器卡片，突出余额数字。
- **Key 选择器、按钮、开关**：统一使用蓝色主题色。
- **空状态**：配合鲸鱼娘插画（如微笑、挥手），增强品牌感。
- **字体与图标**：保持现有 Typography 与圆角系统，图标可统一为蓝色描边风格。

## 3. 桌面小部件

- **尺寸**：保持 2×1（可拉伸）。
- **背景**：使用「鲸鱼娘正在睡觉」的图案作为小部件背景。
- **布局**：
  - 图案占满小部件背景，透明度适中，保证文字可读。
  - 余额文字覆盖在图案上方，使用白色或深蓝描边白色字体。
  - 刷新按钮放在右侧，使用白色或与背景协调的蓝色。
- **氛围**：温暖、安静、治愈，暗示「后台默默守护余额」。

## 4. 约束

- 不使用霓虹、玻璃拟态、大面积高饱和渐变等 AI 模板风效果。
- 保持功能优先，图案不过度遮挡余额、key 名称、刷新按钮等关键信息。
- 暗色模式下，图标与页面蓝色需做相应亮度调整，保证可读性。
